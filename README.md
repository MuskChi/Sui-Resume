## üöÄ Introduction

**Sui Resume** is a decentralized resume platform built on the **Sui Blockchain**. It allows professionals to create, store, and share their resumes securely and immutably.

By leveraging the **Walrus Protocol** for decentralized storage, Sui Resume ensures that your professional history is preserved permanently and remains under your full control, free from centralized interference.

## ‚ú® Key Features

-   **üîó On-Chain Identity**: Mint your professional profile as a dynamic Object on the Sui blockchain.
-   **üêò Decentralized Storage**: High-fidelity assets (profile photos, portfolios) and rich text bios are stored securely on the Walrus Protocol, ensuring data permanence without bloating the chain.
-   **‚ö° Lightning Fast**: Built on Sui, experience sub-second finality for all profile updates and interactions.
-   **üé® Premium UX**: A sleek, responsive interface built with React and Tailwind CSS, designed for the modern web3 professional.
-   **üîí Privacy & Control**: You own your data. Update, hide, or share your profile with cryptographic proof of ownership.

## üì¶ Installation

### Prerequisites
-   [Node.js](https://nodejs.org/) (v18+)
-   [pnpm](https://pnpm.io/) or [npm](https://www.npmjs.com/)
-   [Sui Wallet](https://chrome.google.com/webstore/detail/sui-wallet/opcgpfmipidbgpenhmajoajmjkkhbngg) browser extension

### Setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/MuskChi/Sui-Resume.git
    cd sui-resume
    ```

2.  **Install dependencies**
    ```bash
    cd frontend
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open in Browser**
    Visit `http://localhost:5173` to start exploring.
